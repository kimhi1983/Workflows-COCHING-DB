### 변경 후 (스킬 파이프라인 적용)
```
당신은 화장품 처방 전문가입니다. 아래 6단계 스킬 파이프라인을 반드시 순서대로 적용하세요.

=== SKILL PIPELINE (6단계) ===

[STEP 1: FORMULATION — 처방 설계]
- Phase 구분, 피부타입별 원료 적합성

[STEP 2: COMPOUND-EXPANSION — 복합성분 전개]
- 전개공식: 구성_INCI_wt% = 복합원료_투입wt% × 구성_INCI_비율(fraction)
- 동일 INCI 합산, 향료 단일 표기

[STEP 3: PRECISION-ARITHMETIC — 정수 연산]
★★★ 가장 중요 ★★★
- int_value = round(wt% × 100)
- 밸런스 역산: aqua_int = 10000 - sum(비Aqua int값)
- 3단계 검증: sum==10000, sum/100==100.00, Aqua역산 일치
- Largest Remainder Method 반올림 보정

[STEP 4: INGREDIENTS — 성분 안전성]
[STEP 5: REGULATION — 규제 적합성]
[STEP 6: QA-VALIDATION — 품질 검증]
```

## 5. 변경 상세 — 파싱 & 검증 (🔄 처방 파싱 & 검증)

### 변경 전
```javascript
let totalWt = 0;
guide.phases.forEach(ph => {
  ph.ingredients.forEach(ing => { totalWt += parseFloat(ing.wt_percent) || 0; });
});
wt_valid: Math.abs(totalWt - 100) < 0.1   // 소수점 검증만
```

### 변경 후
```javascript
let totalInt = 0;
let totalWt = 0;
guide.phases.forEach(ph => {
  ph.ingredients.forEach(ing => {
    const wt = parseFloat(ing.wt_percent) || 0;
    totalInt += Math.round(wt * 100);    // 정수 검증 추가
    totalWt += wt;
  });
});

// 3단계 검증
const intValid = totalInt === 10000;
const wtValid = Math.abs(totalWt - 100) < 0.05;  // 허용 오차 축소
const isValid = intValid || wtValid;

// 스킬 메타데이터 자동 삽입
guide.skill_pipeline = {
  version: "v2.3-skill",
  steps: ["formulation","compound-expansion","precision-arithmetic",
          "ingredients","regulation","qa-validation"],
  compound_expanded: hasCompound,
  inci_list_generated: hasInciList,
  precision_int_sum: totalInt,
  precision_valid: isValid
};
```
