# REGULATION SKILL — 규제 적합성 검토

> COCHING AI 가이드처방 파이프라인 Step 5/6
> 한국 식약처 / EU / FDA 화장품 규제 준수 여부 검증

## 1. 트리거 조건
- ingredients 안전성 분석 완료 후 자동 실행
- 처방 전체에 대한 규제 적합성 종합 검토

## 2. 규제 검토 범위

### 2.1 한국 식약처 (MFDS)
```
화장품법 및 화장품 안전기준 등에 관한 규정:

1. 배합 금지 원료 확인 (별표 1)
2. 배합 한도 원료 확인 (별표 2)
   - 자외선 차단제: 성분별 최대 배합 한도
   - 방부제: 성분별 최대 배합 한도
   - 타르 색소: 허용 목록 및 부위별 제한
3. 기능성 화장품 심사
   - 자외선 차단 제품: SPF/PA 표시 기준
   - 미백/주름개선: 고시 원료 및 농도
4. 전성분 표시 기준
   - 함량 내림차순 (1% 초과 성분)
   - 1% 이하 성분은 순서 무관
   - 착향제: "향료"로 통합 표기 가능
5. 나노 원료 표시
   - 나노 형태 사용 시 성분명 뒤에 "(나노)" 표기
```

### 2.2 EU (EC Regulation 1223/2009)
```
1. Annex II: 금지 성분 목록 (1,600+ 종)
2. Annex III: 제한 성분 (배합 한도, 사용 조건)
3. Annex IV: 허용 색소
4. Annex V: 허용 방부제
5. Annex VI: 허용 자외선 차단제
6. 알레르기 유발 향료 26종 별도 표기 의무
7. 나노 원료: 6개월 전 CPNP 신고 + [nano] 표기
8. CMR 물질 (발암/변이원성/생식독성): 원칙 금지
```

### 2.3 FDA (미국)
```
1. OTC Sunscreen Monograph: 허용 자외선 차단 성분 및 한도
   - Zinc Oxide ≤ 25%
   - Titanium Dioxide ≤ 25%
   - 유기자차: 성분별 한도 (Homosalate ≤15%, Octinoxate ≤7.5% 등)
2. Color Additives: 허용 목록 (21 CFR 73, 74, 82)
3. 금지/제한 성분 (21 CFR 700)
4. GRAS (Generally Recognized As Safe) 원료
```

## 3. 규제 위반 등급

| 등급 | 조치 |
|------|------|
| CRITICAL | 배합 금지 성분 감지 → 즉시 제거, 처방 재설계 |
| HIGH | 배합 한도 초과 → 농도 조정 필수 |
| MEDIUM | 표기 의무 누락 → 전성분 표기 수정 |
| LOW | 권고사항 (나노 표기, 알레르기 표기 등) |

## 4. 자외선 차단 제품 특별 규정
```
한국:
  - SPF50+ 표시: 실측 SPF ≥ 50.5
  - PA++++ 표시: PFA ≥ 16
  - 내수성 표시 기준: 40분/80분 수욕 후 SPF 유지율
  - 스프레이형: 흡입 안전성 (나노 TiO2 스프레이 제한)

EU:
  - UVA 방어력: UVA PF ≥ SPF의 1/3
  - Critical Wavelength ≥ 370nm (Broad Spectrum)
  - SPF 50+ 최대 표시 (SPF 60 등 표기 불가)
```

## 5. 출력
- 규제 적합성 판정: PASS / FAIL
- 위반 사항 목록 (등급별)
- 표기 사항 체크 (전성분, 나노, 알레르기 향료)
- 국가별 수출 가능 여부
