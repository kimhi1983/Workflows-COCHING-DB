# QA-VALIDATION SKILL — 처방 품질 검증

> COCHING AI 가이드처방 파이프라인 Step 6/6 (최종)
> 17개 자동 체크 + 이중 문서 교차 검증 + 최종 출력 승인

---

## 1. 트리거 조건
- regulation 규제 검토 통과 후 최종 실행
- 모든 처방은 이 스킬을 통과해야만 출력 가능

## 2. 17개 자동 체크리스트

### 수치 검증 (5항목)
```
[ ] Q1. 정수합 = 10000 (처방서 기준)
[ ] Q2. 정수합 = 10000 (전성분 기준)
[ ] Q3. 처방서 wt%합 = 100.00%
[ ] Q4. 전성분 wt%합 = 100.00%
[ ] Q5. 밸런스(Aqua) 역산 교차 검증 일치
```

### 복합성분 검증 (4항목)
```
[ ] Q6. 모든 COMPOUND 원료의 구성 비율 합계 = 1.000
[ ] Q7. 복합 전개 후 동일 INCI 합산 완료
[ ] Q8. 복합성분 내 Aqua를 밸런스 역산에서 제외 처리
[ ] Q9. Largest Remainder Method 반올림 보정 적용 (필요 시)
```

### 성분 검증 (4항목)
```
[ ] Q10. 배합 금지 성분 없음 (한국/EU/FDA)
[ ] Q11. 배합 한도 초과 성분 없음
[ ] Q12. 피부타입별 부적합 성분 없음
[ ] Q13. 성분 간 비호환 조합 없음
```

### 문서 검증 (4항목)
```
[ ] Q14. 처방서(제조용) 문서 생성 완료
[ ] Q15. 전성분(규제용) 문서 생성 완료
[ ] Q16. 전성분 함량 내림차순 정렬 (1% 초과 기준)
[ ] Q17. 향료 단일 항목 표기 처리
```

## 3. 이중 문서 교차 검증

```
처방서 ↔ 전성분 교차 체크:

1. 총 wt% 일치 확인
   처방서 합계 == 전성분 합계 == 100.00%

2. 성분 누락 확인
   처방서의 모든 투입 원료가 전성분에 반영되었는가?
   (COMPOUND → 전개 INCI로, SINGLE → 그대로, BALANCE → 역산값)

3. 합산 정확성
   처방서에서 동일 INCI가 여러 원료에 포함된 경우
   전성분에서 정확히 합산되었는가?
```

## 4. 판정 기준

| 결과 | 조건 | 조치 |
|------|------|------|
| **PASS** | 17개 전부 통과 | 최종 출력 승인 |
| **WARN** | Q10-Q13 중 CAUTION 존재 | 경고 표기 후 출력 |
| **FAIL** | Q1-Q9 중 하나라도 실패 | 해당 스킬로 돌아가 재처리 |
| **BLOCK** | Q10 CRITICAL 위반 | 처방 재설계 필수 |

## 5. FAIL 시 에스컬레이션

```
Q1-Q5 실패 → precision-arithmetic.md 재실행
Q6-Q9 실패 → compound-expansion.md 재실행
Q10-Q13 실패:
  CRITICAL → formulation.md 재설계
  HIGH → 해당 성분 농도 조정
  MEDIUM/LOW → 표기 수정 후 재검증
Q14-Q17 실패 → 문서 재생성
```

## 6. 품질 검사 항목 (제조 후)

처방서에 포함할 품질 검사 기준:

```
물리 검사:
  - pH 측정 (제품별 적정 범위)
  - 점도 측정 (Brookfield, 온도/스핀들/rpm 명시)
  - 외관 검사 (색상, 균일성, 분리 여부)
  - 비중 측정

미생물 검사:
  - 총 호기성 생균수 ≤ 500 CFU/g (또는 mL)
  - 특정 미생물 부재 확인 (대장균, 녹농균, 황색포도상구균)

안정성 시험:
  - 가속 안정성: 40℃/75%RH, 3개월
  - 동결-해동: -10℃ ↔ 25℃, 3사이클
  - 광안정성: ICH Q1B 기준
  - 원심분리: 3000rpm, 30분

중금속:
  - 납 ≤ 20ppm, 비소 ≤ 10ppm, 수은 ≤ 1ppm, 안티몬 ≤ 10ppm

제품별 추가:
  - 자외선 차단: SPF/PA in-vivo 시험, 입도 분석
  - 색조: 색차 측정 (ΔE)
  - 민감성 제품: 패치 테스트
```

## 7. 최종 출력 형식

QA 통과 시 최종 출력에 포함:
```
1. 처방서 (제조용) — Phase별 투입 원료, 온도, 제조 순서
2. 전성분 (규제용) — INCI 내림차순, 합산 완료
3. 제조 공정 — 단계별 상세 절차
4. 품질 검사 기준 — 물리/미생물/안정성/중금속
5. 처방 특징 요약 — 핵심 컨셉, 차별점
6. QA 검증 결과 — 17항목 체크 + 3단계 수치 검증
```
